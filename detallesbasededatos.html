BASE DE DATOS DISTRIBUIDA COVID-19.

En este proyecto se emplearon los datos de COVID-19 de la página oficial del gobierno de México al 1 de junio de 2020 y se creo una base de datos como de igual manera en su forma distribuida para el análisis y consulta de la misma utilizando SQL Server.
